# Google Cloud Services

This project leverages the following Google Cloud services:

*   **Firebase Hosting:** For hosting the web application.
*   **Firebase Firestore:** As a real-time database for storing cryptographic signatures and public keys.
*   **Firebase Storage:** For storing associated media files.
*   **Vertex AI:** The Gemini Live API is part of the Vertex AI platform, which provides access to Google's generative AI models.
